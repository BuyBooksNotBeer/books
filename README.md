# A modernized Java E*Trade Market Data client

SpringBoot 4.0 and Java 26 acess to E*Trade account and market data API wrapped in a RestController to conveniently pull qoute data and option prices across all expirations.


## Environment Variables

To run this project, you will need to add the following environment variables to the application.properties file or passed in through as JVM parameters or program arguements

`consumer.key`

`consumer.secret`
